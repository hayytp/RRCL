Roblox Ranked Challenges List

This list focuses on finding the HARDEST CHALLENGE in (almost) every game on Roblox and ranking them

List Info

List Rules

  • No Private Servers (Unless they're free)
  • No Hacks / Exploits / Cheats
  • Must submit proof of completion on the Submit Tab of the website or in the #submissions channel in our Discord Server
  • All form of proof MUST be unedited and MUST have the player's Roblox Username

Challenge Requirements

  • No "Grind to Level..." type of challenges (e.g., Level 1000 in Evade)
  • No "Grind to... Ingame Currency/Score" challenges (e.g., 1M Knobs in Doors)
  • No "Meet The Owner/Dev" type of challenges
  • No challenges that require Robux or Roblox Premium to complete
  • No "Get on Leaderboards" type of challenges
  • No "Stay in the Server for..." challenges (e.g., AFK for 24 Hours)
  • Challenges don't necessarily have to be Badges
  • If the Creator of a game asks for their challenge to be removed, it will be removed
  • YOU can suggest a challenge in our Discord Server if it should replace one on the list!
